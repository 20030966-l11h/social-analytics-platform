This folder is static HTML and CSS only—no server required for the marketing site. Host it anywhere; run the Django app separately when you need login and data.

1

Host this landing page

Upload the public_site folder to Netlify, GitHub Pages, Vercel, or any static host. Set the published directory to public_site if your repository contains the whole project.

2

Preview on your computer

From inside public_site, start a simple file server:

python3 -m http.server 8080

Then open http://127.0.0.1:8080/ in your browser.

3

Point visitors to your live app (optional)

Edit index.html and set the content attribute on the <meta name="ssmap-app-url" content="" /> tag to your deployed Django URL. The “Open app” button will then send people straight to your running instance.
